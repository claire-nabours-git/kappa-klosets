# kappa closets 

I built Kappa Klosets, a peer-to-peer marketplace for my UCLA sorority chapter. Greek life involves constant exchanges like formal dresses, raid outfits, furniture, and subleases, but everything was scattered across group chats with no real structure. I created a centralized platform where members can list items, make offers, message each other, and coordinate pickups in one place.

The platform includes real-time listings, a full offer and negotiation flow, a built-in messaging system with unread alerts, and automatic notifications at every stage of a transaction. It is fully mobile-optimized, supports guest browsing for non-members, and is live and actively used within the chapter. Built using React, Firebase, and CSS Modules.
